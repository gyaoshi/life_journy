动画系统集成演示

这个演示展示了如何将动画系统集成到人生旅程游戏中，实现出生动画与游戏计时器的协调控制。

集成要点

动画系统与主游戏的集成包含以下关键功能：

1. 游戏计时器控制

// 设置计时器控制回调 animationEngine.onGameTimerPause = () => { gameTimer.pause(); console.log('游戏计时器已暂停'); }; animationEngine.onGameTimerResume = () => { gameTimer.resume(); console.log('游戏计时器已恢复'); };

2. 出生动画播放

// 播放出生动画 async function startGame() { await animationEngine.playBirthAnimation(); // 动画完成后，角色已显现，计时器已恢复 startGameplay(); }

3. 动画状态监控

// 监控动画状态 function updateGameState() { const animation = animationEngine.currentAnimation; if (animation) { updateUI(animation.currentPhase, animation.currentTime); } }
